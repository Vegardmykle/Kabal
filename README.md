# Kabal

Dette prosjektet er et program for å spille kabal (solitaire), utviklet i java. Her kan du nyte klassisk kabal på din datamaskin, med enkel grafikk og brukervennlig grensesnitt.

## Funksjoner

- Klassisk kabalspill
- Enkel og intuitiv brukeropplevelse
- Mulighet for å starte nytt spill og se poengsum

## Kom i gang

1. Last ned eller klon prosjektet:
   ```bash
   git clone https://github.com/Vegardmykle/Kabal.git
   ```
2. Bygg og kjør med Java:
   ```bash
   javac Kabal.java
   java Kabal
   ```
